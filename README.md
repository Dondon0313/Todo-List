# 📝 Todo List 專案 (Vue 3 + TypeScript)

這是一個使用 Vue 3 + TypeScript 開發的 Todo List 專案，透過 Pinia 管理狀態，並使用 localStorage 儲存資料。

---

## 🚀 使用技術

- ✅ Vue 3 (Composition API)
- ✅ TypeScript
- ✅ Pinia（狀態管理）
- ✅ LocalStorage（本地資料儲存）
- ✅ Tailwind CSS（UI 美化）

---

## 🔧 功能介紹

- ✅ 新增代辦事項
- ✅ 勾選完成狀態
- ✅ 刪除待辦
- ✅ 本地儲存 Todo 清單（localStorage）
- ✅ 使用 Pinia 管理資料狀態
- ✅ 使用 Tailwind CSS 美化 UI
